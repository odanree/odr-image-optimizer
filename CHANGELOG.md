# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-15

### Added
- Initial release of Image Optimizer plugin
- Smart image compression engine with multiple compression levels
- Support for JPEG, PNG, GIF, and WebP formats
- Automatic WebP conversion and browser-compatible fallbacks
- Native lazy loading with Intersection Observer polyfill
- Bulk image optimization with progress tracking
- Admin dashboard with real-time statistics
- Complete REST API for programmatic access
- Custom database tables for optimization history
- Advanced settings page with granular controls
- Auto-optimization on image upload (optional)
- Compression ratio tracking and savings calculator
- Optimization error logging and recovery
- WordPress coding standards compliance
- Comprehensive documentation and developer guides

### Features
- Image compression with quality preservation
- WebP conversion (up to 35% smaller files)
- Lazy loading for improved page performance
- Bulk operations for large image libraries
- REST API endpoints
- Database-backed history and statistics
- Admin UI for easy management
- Settings for customization

### Security
- Nonce verification on all forms
- Capability checks (manage_options)
- Input sanitization and validation
- Output escaping
- SQL prepared statements
- Direct file access prevention
